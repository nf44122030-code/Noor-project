## Steps after adding photos:
1. Place photos in this folder (`assets/experts/`)
2. The app already registers this directory in `pubspec.yaml`
3. Run `flutter pub get` and hot-restart the app
4. Photos will appear automatically in the Expert Sessions screen

> **Note:** Until photos are added, the app shows a styled initials avatar as placeholder.
